# ⚽ Olasılık Stadyumu

Türk Süper Lig takımları arasında olasılık tabanlı interaktif maç simülasyonu.

## 🎮 Özellikler

- Trabzonspor, Fenerbahçe, Beşiktaş ve Galatasaray arasında maç seçimi
- Tahmin spektrumu — topu sürükleyerek kazanma olasılığı tahmini
- Gerçek zamanlı maç simülasyonu (oyuncu hareketleri, top fiziği, gol animasyonları)
- Maç müziği — maç başlayınca otomatik çalar, bitince durur
- Ses aç/kapa butonu
- N maç simülasyonu istatistikleri

## 📁 Klasör Yapısı

```
olasilik-maci/
├── index.html          # Ana uygulama
├── assets/
│   └── audio/
│       └── mac-muzigi.mp3   # Maç müziği
└── README.md
```

## 🚀 Kullanım

Projeyi klonlayın ve `index.html` dosyasını bir tarayıcıda açın.  
Herhangi bir sunucu gerekmez — tamamen istemci taraflı çalışır.

```bash
git clone https://github.com/kullanici-adi/olasilik-maci.git
cd olasilik-maci
# index.html dosyasını tarayıcıda aç
```
